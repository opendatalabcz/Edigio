Česká verze:

Byla vytvořena žádost o změnu Vašeho telefonního čísla.
Prosíme, zadejte následující kód, čímž potvrdíte, že byla žádost opravdu vytvořena Vámi:

Pokud jste o žádost nezažádali, tuto zprávu prosím ignorujte.

English translation:

Change of your phone number has been requested.
Please, enter the following code to confirm that the request was issued by you:

If you didn't create the request, please, ignore this message.
